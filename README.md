LibS2kMounts
============

A library to convert internal MountId (e.g. returned by GetActionInfo) to SpellId.

Synopsis:
---------
    local S2KMounts = LibStub("LibS2kMounts-1.0")

    spellId = S2KMounts:GetSpellIdByMountId(mountId)
